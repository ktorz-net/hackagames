# HackaGames - Several Shell-Based Hackable Games

**HackaGames** aims to be an open game engine dedicated to the development of Artificial Intelligence (AI) based on Operational Research (OR) technics.

This repo regroups first shell-based games on the **HackaGames** frameworks: [hacka(py)](https://www.github.com/ktorz-net/hacka-py) for _Pyhon_ implementation and [hacka(lib)](https://github.com/ktorz-net/hacka-lib) for _C_ implementation (not well advanced righ now...)

- Documentation: [ktorz-net.github.io/hackagames](https://ktorz-net.github.io/hackagames).


## Presented Games:

- **Py421** (Python): A very simple one player dice game to get the concept of AI implementation with _HackaGames_.

## Get Started

The Python version of the **HackaGames** engine is shared through [pypi.org](https://pypi.org/project/hacka/) and can be installed with `pip install hacka`. 
It is possible to install it from this repository to use the on-developpement version.

```shell
git clone https://www.github.com/ktorz-net/hacka-py
pip install ./hacka-py
```
